# Space Missions Data Analysis - Tableau Dashboard

## Overview
This project showcases a comprehensive analysis of global space missions. The interactive dashboard visualizes trends in space exploration, success rates, and mission distribution across different nations/organizations.

## Dashboard Link
**[View Interactive Dashboard on Tableau Public](https://public.tableau.com/app/profile/sukhjeet.singh5355/viz/SpaceMissionsDataAnalysis_16831441771950/Worldinfomartion#1)**

## Key Features
- **Mission Success Analysis:** Track the success and failure rates of missions over the last few decades.
- **Geographic Insights:** Visualize how different nations have contributed to space exploration.
- **Trend Evolution:** Identify the shift in space mission frequency from the early space race era to the modern private sector boom.

## Technical Skills Demonstrated
- **Data Cleaning & Preprocessing:** Prepared raw mission data for visualization.
- **Interactive Dashboarding:** Built dynamic views, filters, and maps in Tableau.
- **Data Storytelling:** Translating large datasets into actionable insights for general audiences.
